# Stopwatch

## Description
A simple and interactive **Stopwatch** built using **HTML**, **CSS**, and **JavaScript**.  
It displays hours, minutes, seconds, and milliseconds, and allows users to start, stop, and reset the timer.

---

## Features
- Start, stop, and reset timer  
- Shows **hours:minutes:seconds:milliseconds**  
- Responsive and clean design  
- Buttons change color on hover  

---

## How It Works
### **Start**
- Click the **Start** button to begin timing.  
- Timer updates every 10 milliseconds.  

### **Stop**
- Click the **Stop** button to pause the timer.  

### **Reset**
- Click the **Reset** button to set the timer back to `00:00:00:00`.

## How to Use
1. Open **index.html** in any web browser.  
2. Click **Start** to begin the timer.  
3. Click **Stop** to pause.  
4. Click **Reset** to clear the timer.
